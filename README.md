# d3js-hierarchy
module d3js (2D layout algorithms for visualizing hierarchical data)

## Clone the repo & install
```
git clone https://github.com/suxxus-studies/d3js-hierarchy.git
cd ./d3js-hierarchy
npm install
```
## Usage
npm run dev
